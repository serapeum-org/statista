# Parameters

The `statista.parameters` package provides parameter estimation methods
for statistical distributions using L-moments.

- [L-moments](lmoments.md) — L-moments calculation and core estimation
- [Extreme Value](extreme-value.md) — GEV and Gumbel parameter estimation
- [Normal Family](normal-family.md) — Normal and Generalized Normal estimation
- [Other](other.md) — Gamma, Pareto, Wakeby, Pearson III estimation
