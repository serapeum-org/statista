# Distributions

The `statista.distributions` package provides probability distribution classes
for statistical analysis and extreme value modeling.

- [Facade](facade.md) — Factory class for creating distributions by name
- [Plotting Position](plotting-position.md) — Empirical CDF and return period utilities
- [Gumbel](gumbel.md) — Gumbel (Type I extreme value) distribution
- [GEV](gev.md) — Generalized Extreme Value distribution
- [Exponential](exponential.md) — Exponential distribution
- [Normal](normal.md) — Normal (Gaussian) distribution
